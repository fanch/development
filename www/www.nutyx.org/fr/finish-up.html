Et maintenant ?

Vous voici avec une NuTyX de base fraîchement installée. Si vous connaîssez déjà NuTyX, vous pouvez directement vous diriger sur la documentation du site. Par contre si vous vous demandez comment installer une interface graphique complète, continuez la lecture.

Mise à jour des informations concernant les paquets disponibles

Il faut commencer par renseigner les paquets disponibles à votre nouvelle installation. Un alias est disponible à cet effet: 
check
[sudo] password for thierry:
Vous entrez le mot de passe que vous avez choisi lors de l'installation et la synchronisation démarre: 
/var/lib/pkg/depot/gnome-extra/.PKGREPO
/var/lib/pkg/depot/gnome/.PKGREPO
/var/lib/pkg/depot/mate-extra/.PKGREPO
/var/lib/pkg/depot/mate/.PKGREPO
/var/lib/pkg/depot/kde5-extra/.PKGREPO
/var/lib/pkg/depot/kde5/.PKGREPO
/var/lib/pkg/depot/xfce4-extra/.PKGREPO
/var/lib/pkg/depot/xfce4/.PKGREPO
/var/lib/pkg/depot/gui-extra/.PKGREPO
/var/lib/pkg/depot/gui/.PKGREPO
/var/lib/pkg/depot/cli-extra/.PKGREPO
/var/lib/pkg/depot/cli/.PKGREPO
/var/lib/pkg/depot/base/.PKGREPO

Installation d'une interface graphique et d'un des bureaux disponible

La synchronisation est terminée. Vous êtes à présent prêt à installer le reste des paquets pour obtenir une NuTyX utilisable. Commençons donc par installer le serveur graphique: 
get xorg
[sudo] password for thierry:
Vous entrez le mot de passe que vous avez choisi lors de l'installation et l'installation démarre: 
/var/lib/pkg/depot/gui/xorg/.PKGREPO
  ...
Vous pouvez maintenant choisir votre bureau préféré. Sous NuTyX, vous avez le choix entre: kde5, xfce4, mate, gnome, lxde, openbox, flwm, ratpoison, etc

Bureau disponibles:
Nom du bureau: Commande:
kde5 kde5-extra 
get kde5 kde5-extra
gnome gnome-extra 
get gnome gnome-extra
xfce4 xfce4-extra 
get xfce4 xfce4-extra
mate mate-extra 
get mate mate-extra
lxde 
get lxde
openbox 
get openbox

Quelque soit le choix de votre bureau, vous allez devoir installer le gestionnaire de connection graphique. Vous avez le choix entre lxdm et slim. Pour lxdm, installez: 

get lxdm

Vous installez maintenant les outils qui vous semblent utiles, par exemple: 

get zip unzip xdg-user-dirs gvfs alsa-utils ntfs-3g firefox libreoffice pidgin thunderbird

Si vous utilisez un réseau sans fils ou êtes en mode itinérant, installez le gestionnaire de connections réseaux et son applet: 

get network-manager-applet

Et pour finir

Le plus simple maintenant redémarrez votre machine. Après le redémarrage, vous serez accueilli par le gestionnaire de connection lxdm ou slim. Pour ce dernier, pressez la touche F1 pour choisir le bureau adéquat. 
sudo reboot

Conclusion

Je vous invite à prendre connaîssance des commandes de base pour que vous puissiez allez plus loin dans votre installation.
