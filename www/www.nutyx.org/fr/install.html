Installation classique de NuTyX sur une seule partition

Ces captures d'écrans vous montrent une installation classique sur une seule partition en mode bios. Le disque dur est de 8 goctets. La première partition est la partition d'échange et sa taille est de 2Goctets Tout le reste de la place est disponible pour la partition sur laquelle est installée NuTyX.

La taille mémoire minimum recommandée est de 1 Goctet.

L'iso est transférée sur CD. Vous pouvez évidement utiliser une clé USB si votre PC le permet.

Le disque est totalement vierge et n'a jamais été formaté.

Sommaire

  • Démarrer la clé / le CD sur la(le)quel se trouve l'ISO
  • Création de la partition (optionnel)
  • Formatage de la partition (optionnel)
  • Installation de NuTyX
  • Configuration de NuTyX
  • Première prise en main

Démarrer la clé / le CD sur la(le)quel se trouve l'ISO

Création de la partition (optionnel)

La création de la partition n'est pas indispensable. Si vous avez déjà une partition disponible vous pouvez directement passer au formatage de la partition.

Formatage de la partition (optionnel)

Le formatage de la partition n'est à faire que si vous venez de la créer ou vous souhaitez effacer l'intégralité de son contenu. Si au contraire vous souhaitez conserver certains dossiers, vous pouvez alors supprimer uniquement les dossiers opt, bin, etc, root, run, sbin, usr et var qui se trouvent sur la partition de destination. Cette méthode permet de conserver vos données personnelles par exemple. Si la partition de destination est vierge et déjà formatée, vous pouvez directement passer à l'installation de NuTyX.

Installation de NuTyX

L'installation de NuTyX se fait sans connection réseau.

Elle ne prend que quelques secondes.

Configuration de NuTyX

Une fois installée, NuTyX se lance immédiatement sans avoir besoin de redémarrer la machine.

La configuration est très directe: réglage de la carte réseau, de la date, de l'heure et création du premier utilisateur.

Une fois la configuration terminée, vous vous connectez en utilisant le nom d'utilisateur que vous avez créé à l'instant.

Première prise en main

Je vous invite à prendre connaîssance des quelques commandes nécessaires pour que vous puissiez terminer votre installation avec une interface graphique par exemple.
